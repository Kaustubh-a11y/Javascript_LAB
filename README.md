The entire collection is powered by a central UI library and developer tool suite:

## 🚀 Running the Practicals
Practicals can be viewed directly in the browser. Since they use pure modern standard JS and local assets, you can run them in two ways:

1. **Local File Scheme**:
   Open the `index.html` file of any Practical directly in Google Chrome.
   
2. **Local HTTP Server**:
   Run a simple python web server inside the directory:
   ```bash
   python -m http.server 8080
   ```
   Navigate to: `http://localhost:8080/`
