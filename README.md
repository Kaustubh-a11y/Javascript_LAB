A collection of **10 JavaScript laboratory practicals** built using **HTML, CSS, and Vanilla JavaScript**. Each practical demonstrates different JavaScript concepts through interactive web-based exercises with a consistent UI and responsive design.

---

## 📁 Project Structure

The practicals share common resources:

- **`shared.css`** – Common styles, reusable UI components, and layout utilities.
- **`console-emulator.js`** – Displays JavaScript console output (`console.log`, `console.error`, `console.table`, etc.) directly on the webpage for easier debugging.

---

## 📚 Practicals

### 01. JS Execution & Console Methods
- **Location:** `Practical 01/`
- Demonstrates inline, internal, and external JavaScript along with console methods.

### 02. Billing Calculator
- **Location:** `Practical 02/`
- Uses variables, template literals, objects, arrays, and basic calculations to generate a bill.

### 03. Student Grading System
- **Location:** `Practical 03/`
- Calculates grades, GPA, and validates marks using conditional statements.

### 04. Palindrome & Closure Checker
- **Location:** `Practical 04/`
- Checks whether a string is a palindrome and demonstrates JavaScript closures.

### 05. Cart Calculator
- **Location:** `Practical 05/`
- Shopping cart with quantity controls, total calculation, and discount logic.

### 06. String & Regex Utility
- **Location:** `Practical 06/`
- Performs string operations, regex matching, and input validation.

### 07. Task Planner
- **Location:** `Practical 07/`
- Simple to-do application with filtering, priorities, and progress tracking.

### 08. Gym Admission Portal
- **Location:** `Practical 08/`
- Registration form featuring validation, BMI calculator, dark mode, and user preferences.

### 09. Storage Preference Manager
- **Location:** `Practical 09/`
- Demonstrates the use of `localStorage` and `sessionStorage`.

### 10. Dynamic JSON Data Grid
- **Location:** `Practical 10/`
- Displays local JSON data with search, sorting, and filtering.
